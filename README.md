# Bot_for_Acquaintance

Простой Telegram-бот для знакомства студентов с анкетами, лайками, дизлайками и перезаполнением профиля.

---


## ⚙️ Настройка

В файле `.env` укажите:

```env
BOT_TOKEN=<ваш_токен_бота>
LOG_LEVEL=INFO
```

---

## 🧩 Структура проекта

- `bot.py` — точка входа, запускает Telegram-бота и регистрирует хендлеры
- `config.py` — загружает `BOT_TOKEN` и `LOG_LEVEL` из `.env`
- `bd/db.py` — база данных SQLite: пользователи, анкеты, лайки
- `handlers/handlers.py` — основная логика бота и обработка команд
- `keyboard/keyboar.py` — клавиатуры для меню и анкет

---

## 📝 Как работает бот

- `/start` — начало работы, показывает кнопки регистрации или главное меню
- `Регистрация` — сохраняет Telegram-никнейм пользователя
- `Моя анкета` — показывает текущую анкету или запускает заполнение
- `Знакомиться` — показывает случайную анкету другим пользователям
- В анкете доступны кнопки:
  - `❤️ Лайк`
  - `👎 Дизлайк`
  - `⚠️ Жалоба`
- При создании анкеты сохраняется фото и текст в базе

---

## ✅ Особенности

- Поиск показывает анкету только нужного пола, если выбран `Мужской` или `Женский`
- При выборе `Без разницы` — показываются любые анкеты
- Если пользователь уже заполнил анкету, при нажатии `Моя анкета` показываются две кнопки:
  - `Главное меню`
  - `Заполнить анкету заново`
- При повторном заполнении:
  - старая анкета сбрасывается
  - все предыдущие лайки/дизлайки/жалобы удаляются
  - пользователь начинает заново

---

## ▶️ Запуск

```bash
python bot.py
```

---

## 📌 Примечания

- База хранится в `db.db`
- Пользователи могут лайкать других сразу, и при этом получатель всегда получает уведомление `Вас лайкнул @никнейм`
